## API (Front-End)
*Descrição:*
- Este projeto é uma aplicação de um marketplace voltado para a revenda de jogos eletrônicos.
---
- *Tecnologias*
    - Framework
        - Node (20.9.0)
        - npm (10.2.0)
        - yarn (1.22.21)
    - Linguagem
        - Java Script 
            - React Native
            - Type Script 
    - App IOS
        - Expo GO
